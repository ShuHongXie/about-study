### 结构体 struct

释义：不同类型数据夬成的组合型的数据结构，被称为结构体。

```c
// 定义
struct 结构体名 {
  成员表列
} 变量名表列
// 例子
struct student {
  int num;
  char name[20];
  char sex;
  char addr[20];
} student1, student2 = { 10101, "shuhongxie", 'M', "guangzhou" };
```

### 结构体数组

```c
struct person {
  char name[20];
  int count;
} leader[3] = {"Li",0,"Zhang",0,"Fun",0};
```
