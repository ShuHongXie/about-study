### 小程序性能衡量指标

1. 首次渲染时间(onLoad 到 onShow)：页面首次渲染完成，表示页面已经加载可以和视图层交互；

2. 首屏加载时间(onLoad 到 this.setData 回调完)：首屏部分的内容已经渲染完成，用户可以看到首屏；

### 登录流程

1. 小程序启动时（app.js 的 onLaunch）或用户点击 “登录” 按钮时，调用 wx.login() 获取临时登录凭证 code（有效期 5 分钟，只能用一次）后调用后端接口
2. 后端调用微信接口换取用户标识：后端拿到前端传的 code 后，调用微信官方的 code2Session 接口，获取用户唯一标识（openid）和会话密钥（session_key）
3. 后端拿到 openid 后，做两件事：
   关联 / 创建用户：根据 openid 查询数据库，存在则关联已有用户，不存在则创建新用户；
   生成 token：用 JWT / 自定义规则生成业务 token（比如包含用户 ID、过期时间），返回给前端。
4. 前端拿到 token 后，存储在 wx.setStorageSync 中，后续所有需要鉴权的接口，都在请求头中携带 token

### H5状态同步

一、小程序端流程
用户已登录，小程序持有业务 token
调用后端接口 → 申请临时 ticket
后端生成 ticket（5 分钟、一次性）
存入 Redis：标记未使用、关联用户、绑定页面
小程序打开 web-view，url 只携带：
ticket
timestamp
appid
⚠️ 不携带真实 token
监听 H5 发来的消息，做环境校验
二、H5 端流程
从 url 解析 ticket
立刻清掉 url 里的参数（history.replaceState）
通过 JsBridge 让小程序校验环境是否合法
防止被外链打开

合法 → 拿 ticket 请求后端换取 H5 专用 token
后端校验 ticket：
未使用
未过期
来源合法

校验通过 → 标记 ticket 已使用

后端返回 H5 短期 token（2 小时）
H5 存储到 sessionStorage，配置请求头
自动实现：
token 过期自动刷新
刷新失败 → 通知小程序重登

### webview预加载

```js
// 小程序全局App.js（提前初始化预加载池）
App({
  onLaunch() {
    // 初始化H5预加载池（存储隐藏的web-view上下文）
    this.h5PreloadPool = new Map();
  },

  // 预加载指定H5链接
  preloadH5(h5Key, h5Url) {
    // 避免重复预加载
    if (this.h5PreloadPool.has(h5Key)) return;

    // 1. 生成预加载web-view的临时页面路径（需在app.json注册）
    const preloadPagePath = `/pages/h5-preload/h5-preload`;
    // 2. 打开隐藏的预加载页面（不显示给用户）
    wx.navigateTo({
      url: `${preloadPagePath}?h5Key=${h5Key}&h5Url=${encodeURIComponent(h5Url)}`,
      animationType: "none", // 无动画，避免用户感知
      success: (res) => {
        // 3. 存储页面上下文到预加载池
        this.h5PreloadPool.set(h5Key, {
          pageContext: res,
          h5Url,
          createTime: Date.now(),
        });
      },
    });

    // 4. 预加载池清理策略（避免内存泄漏，10分钟未使用则销毁）
    setTimeout(
      () => {
        if (this.h5PreloadPool.has(h5Key)) {
          const poolItem = this.h5PreloadPool.get(h5Key);
          poolItem.pageContext?.close(); // 关闭隐藏页面
          this.h5PreloadPool.delete(h5Key);
        }
      },
      10 * 60 * 1000,
    );
  },
});

// 预加载页面（pages/h5-preload/h5-preload）的wxml
<web-view
  src="{{h5Url}}"
  style="width: 0; height: 0; opacity: 0;" // 隐藏容器
  bindload="handleH5Load"
></web-view>;

// 预加载页面的js
Page({
  onLoad(options) {
    this.setData({
      h5Key: options.h5Key,
      h5Url: decodeURIComponent(options.h5Url),
    });
  },
  // H5加载完成后标记状态
  handleH5Load() {
    getApp().h5PreloadPool.get(this.data.h5Key).loaded = true;
  },
});

// 业务页面调用预加载（比如首页onLoad时）
Page({
  onLoad() {
    // 预加载高频访问的H5页面（比如工单详情）
    const ticketDetailH5Url = `https://你的H5域名/ticket-detail.html?xxx`;
    getApp().preloadH5("ticketDetail", ticketDetailH5Url);
  },
  // 用户点击打开H5时
  openH5() {
    const poolItem = getApp().h5PreloadPool.get("ticketDetail");
    if (poolItem && poolItem.loaded) {
      // 预加载完成，直接打开目标web-view页面
      wx.navigateTo({
        url: `/pages/web-view/web-view?h5Url=${encodeURIComponent(poolItem.h5Url)}`,
      });
    } else {
      // 预加载未完成，正常打开（兜底）
      wx.navigateTo({
        url: `/pages/web-view/web-view?h5Url=${encodeURIComponent(ticketDetailH5Url)}`,
      });
    }
  },
});
```
